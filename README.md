# PRAESIDIUM

**Praesidium** is a large-scale factory automation and planetary defense game currently in active development.

Build automated production lines, expand across multiple continents, establish global logistics networks, and activate the **Praesidium Shield** to protect an entire planet from an ongoing alien invasion.

---

## About

Praesidium is an automation game where production, logistics, energy, exploration, and defense are deeply connected.

Instead of simply building the biggest factory possible, the player's ultimate objective is to construct a planetary defense network capable of stopping an endless extraterrestrial threat.

Every production line, railway, drone fleet, and power grid contributes toward a single long-term goal.

---

## Planned Features

* Large-scale factory automation
* Multi-continent procedural world
* Advanced logistics systems
* Conveyor belts, trains, and drone transportation
* Physical logic system (AND, OR, NOT, XOR...)
* Programmable drones with scripting support
* Dynamic enemy invasions
* Planet-wide defense infrastructure
* Persistent world simulation
* Heavy industrial atmosphere inspired by Victorian engineering

---

## Current Status

Praesidium is currently in the early stages of development.

The project is focused on designing the core gameplay systems and building the simulation architecture before content production begins.

Features and mechanics are subject to change throughout development.

---

## Technology

* **Language:** C++
* **Graphics Library:** Raylib

---

## Project Goals

The long-term vision of Praesidium is to create an automation game where players gradually evolve from managing a single factory into controlling an entire industrial civilization.

Production, logistics, defense, automation, and planetary infrastructure should all become parts of one interconnected system.
